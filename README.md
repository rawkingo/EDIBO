#### 1. Terminal (hot-keys)
Ctrl+Alt+T - open **Terminal**  
Ctrl+Shift+T - open a new tab in **Terminal**  
Ctrl+Shift+Q or ALT+F4 - close **Terminal**  
Ctrl + - Zoom In **Terminal**  
Ctrl - - Zoom Out **Terminal**
